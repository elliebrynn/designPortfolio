Jazz In July Poster Series

Typography & Poster Design

The Jazz In July Poster Series was a challenge designed to practice textual hierarchy in my Typography course at Indiana University. The concepts are for a music event hosted on IU Bloomington's campus. The expectation for this project was to create four unique poster designs using the event information provided and experimenting with textual hierarchy. The first design was to be created using a single typeface of our choosing at the same point size using only black text. The next design had to be a single typeface as well, but we could use a variety of point sizes to create hierarchy. The third design was permitted to use color, and the final design was free-reign, but I appreciated the constraints enough to create an additional design within the same framework.

My understanding of textual hierarchy broadened to include more that just point size because I understood that hierarchy can be created in diverse ways. I implemented spacial concepts like the Gestalt principles of Similarity and Proximity, as well as varying font weights to create order that did not yet exist visually. I also created abstract visual cues that subliminally communicate the subject matter. The color blocking and line work in the fourth and final piece is meant to represent piano keys, symbolizing the musical nature of the event being advertised.
